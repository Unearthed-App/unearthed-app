# Unearthed - Transform Your Reading Into Connected Knowledge

> **Past Insights, New Revelations**

Unearthed is a comprehensive reading ecosystem that transforms your scattered Kindle and KOReader highlights into a connected knowledge system. Whether you prefer complete privacy with our local app or the full-featured cloud experience, Unearthed helps you capture, organize, and rediscover your reading insights across multiple platforms.

## 🌟 What is Unearthed?

Unearthed bridges the gap between reading and knowledge retention by automatically syncing your highlights and notes from various sources, then intelligently connecting them to create meaningful insights. The platform consists of multiple interconnected components designed to fit different privacy preferences and workflow needs. The simplest setup would be to choose to install the [Unearthed Local](https://unearthed.app/local) app as that is the only thing you would need.

### The Unearthed Ecosystem

- **[Unearthed Local](https://unearthed.app/local)** - Local desktop application (one-time purchase, no subscriptions)
- **[Unearthed Mobile](https://unearthed.app/mobile)** - Progressive Web App (PWA) for Android, works alongside Unearthed Local
- **[Unearthed Online](https://unearthed.app)** - This Next.js web application with AI-powered features and cloud sync
- **Browser Extension** - Automatic Kindle sync from Amazon's Kindle Notebook
- **Obsidian Plugin** - Direct integration with your Obsidian vault
- **KOReader Plugin** - Wireless sync from e-ink devices and KOReader apps

## 🚀 Key Features

### Universal Sync Capabilities

- **Kindle Integration**: Automatically sync all highlights and notes from your Amazon Kindle Notebook
- **KOReader Support**: Wireless sync from KOReader devices and apps
- **Smart Merging**: Combines highlights from the same book across different sources
- **Platform Connectivity**: Sync to Obsidian, Notion, Capacities, and Supernotes

### Intelligent Analysis (Unearthed Online)

- **AI-Powered Insights**: Optional AI analysis to detect blind spots and extract themes
- **Daily Reflections**: Curated highlights delivered via email or daily notes
- **Interactive Graph View**: visualise connections between books, highlights, and tags
- **Book Recommendations**: Discover similar and contrasting viewpoints
- **Chat with Books**: AI-powered conversations with individual books

### Flexibility

- **Two Options**: Choose between cloud-based convenience or complete local
- **Open Source**: All Online components are open source and available on GitHub
- **No Lock-in**: Export your data anytime in multiple formats
- **Custom Templates**: Format how books and highlights appear in your target platforms

## 🔧 Component Breakdown

### Unearthed Local

- **Desktop Application**: Native apps for Windows, Mac, and Linux
- **Complete Privacy**: All data stays on your device
- **One-time Purchase**: No subscriptions or recurring fees
- **Background Sync**: Automatic syncing while running in the background
- **Browser Extension and Obsidian Plugin are not required for [Unearthed Local](https://unearthed.app/local)**

### Browser Extension

- **Chrome & Firefox**: Available on official web stores once installed
- **Auto-sync**: Reads Amazon Kindle Notebook and sends highlights to Unearthed
- **Privacy-focused**: No Amazon credentials stored or transmitted
- **Smart updates**: Only syncs new highlights, avoiding duplicates

### Obsidian Plugin

- **Compatibility**: Works with [Unearthed Online](https://unearthed.app/online)
- **Custom Templates**: Fully customizable formatting for books and highlights
- **Daily Reflections**: Automatic integration with daily notes
- **Tag Management**: Sync tags and create connections in your vault

### Unearthed Mobile

Unearthed Mobile is a Progressive Web App (PWA), designed as a mobile companion to [Unearthed Local](https://unearthed.app/local). It runs on Android and communicates directly with your Local instance over your home network — no cloud account required.

- **Works with Unearthed Local**: Connects to your Local desktop app to browse and read your highlights on the go
- **No subscription required**: Included with Unearthed Local
- **Native feel**: Installable as a PWA on Android for a full-screen, app-like experience
- **RSS, web import, YouTube**: Fetched directly from the device — works on WiFi and mobile data

#### Installing the PWA on Android (Chrome)

1. Open [unearthed.app/mobile](https://unearthed.app/mobile) in **Chrome**
2. Tap the **three-dot menu** (⋮) in the top-right corner
3. Tap **Add to Home screen** (or **Install app** if prompted automatically by a banner)
4. Tap **Add** to confirm
5. The app icon will appear on your home screen — tap it to launch in full-screen mode

> **Tip**: On Android, Chrome may show an automatic "Install app" prompt at the bottom of the screen — tap it to skip the menu steps.

### KOReader Plugin
- **Compatibility**: Works with both [Unearthed Local](https://unearthed.app/local) and [Unearthed Online](https://unearthed.app/online)
- **Wireless Sync**: No need to physically connect devices
- **Cross-platform**: Works on Android, Kobo, Kindle, and other e-ink devices
- **Flexible Setup**: Supports both [Unearthed Online](https://unearthed.app/online) and [Unearthed Local](https://unearthed.app/local)

## 🚀 Getting Started
### View the appropriate Readme: [Unearthed GitHub](https://github.com/unearthed-app)

### For Unearthed Online

1. Create an account at [unearthed.app](https://unearthed.app)
2. Install the browser extension for Kindle sync
3. Connect your preferred platforms (Obsidian, Notion, etc.)
4. Start syncing and exploring your reading insights

### For Unearthed Local

1. Download from [unearthed.app/local](https://unearthed.app/local)
2. Install the desktop application
3. Set up local sync with your devices and platforms
4. Enjoy complete privacy and ownership

### For Unearthed Mobile (PWA)

1. Install [Unearthed Local](https://unearthed.app/local) on your desktop first
2. Open [unearthed.app/mobile](https://unearthed.app/mobile) on your phone
3. Install it as a PWA ( [Android](#installing-the-pwa-on-android-chrome) instructions above)
4. Connect to your Local instance over your home network
